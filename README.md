# Конвертер учебного конфигурационного языка в YAML

## Описание
Инструмент командной строки для преобразования текста из учебного конфигурационного языка в YAML формат. Вариант 2. 

## Возможности
- Поддержка чисел, массивов, констант
- Однострочные и многострочные комментарии
- Проверка синтаксических ошибок
- Рекурсивный парсинг вложенных конструкций

## Использование
```bash
python config_converter.py -i input.config -o output.yaml
